# Coding Conventions
## Overview
+ base on PSR : <https://www.php-fig.org/psr/psr-2/>

## File
+ Tags : `<?php` / `<?=` 태그만 사용. PHP code만 존재하는 파일은 `?>` 태그 생략
+ Encoding : UTF-8로만 작성
+ Extension : html로 파일 저장. 단, common/include 형태는 php로 저장
+ `'` 홀따옴표 사용
+ 들여쓰기 : 2개 스페이스. 탭 사용 안 함
